Bio

I graduted from General Assembly's Web Development Immersive course where I learned many technologies such as HTML, CSS, Twitter Bootstrap, JavaScript, jQuery, MEAN Stack (MongoDB, Express, AngularJS, Node.js), Ruby, Rails, RESTful API Integrations, Heroku, Git and Github.
